AirTap Project - team code 



This folder contains all of our custom code for our tap detection system. We are building on top of the original 3D CNN gesture recognition repo but adding a couple of things.

* mediapipe fingertip tracking
* time series feature extraction like distance velocity
* 1D cnn for tap detection
* fusion between 1d cnn and 3d cnn outputs



Our goal is to detect and count tap gestures in real time.
